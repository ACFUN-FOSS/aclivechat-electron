## 和 ShigemoriHakura/aclivechat 的不同

与<a href="https://github.com/ShigemoriHakura/aclivechat">ShigemoriHakura/aclivechat</a>的不同

- 增加了气泡弹幕样式
- 删除了与 acfun 直播无关的文字和设置项
- 前后端 JavaScript 全栈
- 更精简的 Api 设计
- ws 出错自动重连，理论上不会寄了（我猜，大概
